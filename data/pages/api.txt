====== API ======
OpenNIC maintains an API page as a gateway to various projects. The default page is located at https://api.opennicproject.org/ however there is no official website on that page.

The following resources may be accessed through the API:
  * [[api:geoip]] - ([[https://api.opennicproject.org/geoip/|/geoip/]])
  * [[api:blacklist]] - ([[https://api.opennicproject.org/acl/filter/|/acl/filter/]])
  * [[api:whitelist]] - ([[https://api.opennicproject.org/ip/update/|/ip/update/]])
  * [[api:bindacl]] - ([[https://api.opennicproject.org/acl/bind/|/acl/bind/]])
